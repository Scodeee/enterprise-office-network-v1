# Enterprise Office Network v1 - Network Fundamentals

## 📖 Overview

Enterprise Office Network v1 merupakan simulasi implementasi jaringan perusahaan berskala kecil menggunakan **Cisco Packet Tracer**. Proyek ini dibuat sebagai bagian dari portofolio untuk mempelajari dan mengimplementasikan konsep dasar **Network Administrator** yang umum digunakan di lingkungan perusahaan.

Implementasi difokuskan pada pembangunan jaringan dari nol, mulai dari perencanaan topologi, perancangan IP Address, segmentasi jaringan menggunakan VLAN, konfigurasi DHCP, DNS, hingga pengujian konektivitas antar perangkat.

Repository ini merupakan bagian pertama dari **Enterprise Network Series**, yaitu roadmap pembelajaran yang akan terus dikembangkan menuju implementasi jaringan enterprise dan network security.

---

# 🎯 Project Objectives

Project ini bertujuan untuk:

- Mendesain topologi jaringan perusahaan sederhana.
- Mengimplementasikan segmentasi jaringan menggunakan VLAN.
- Mengkonfigurasi Router-on-a-Stick untuk komunikasi antar VLAN.
- Mengimplementasikan DHCP Server agar setiap host memperoleh IP Address secara otomatis.
- Mengimplementasikan DNS Server untuk menerjemahkan nama domain internal.
- Menghubungkan jaringan internal dengan jaringan Internet menggunakan Static Routing.
- Mendokumentasikan seluruh proses implementasi sebagai standar dokumentasi Network Administrator.

# 🏢 Business Scenario

Sebuah perusahaan startup teknologi memiliki dua lantai kantor yang saling terhubung melalui sebuah Core Switch.

### Lantai 1

- Reception
- Finance
- HR
- Meeting Room

### Lantai 2

- IT Support
- Developer Room
- Manager Room
- Server Room

Selain jaringan kabel, perusahaan juga menyediakan jaringan WiFi yang dapat digunakan oleh perangkat mobile.

Seluruh pengguna harus memperoleh alamat IP secara otomatis dan dapat mengakses layanan internal perusahaan maupun Internet.
